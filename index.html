Question 3: Real-World Application of Tuples Scenario: As part of a meteorological study, you receive daily weather observations as tuples containing temperature, humidity, and wind speed. Your task is to process these observations and convert them into a structured format that can be easily analyzed. Interview Questions: How do tuples compare to lists in terms of mutability and use cases in data processing? Can you provide an example of a situation where converting a tuple to a dictionary would be beneficial? How would you handle missing or unexpected data in the weather observation tuples? Are there any limitations to using tuples for storing and processing structured data? Coding Challenge: Develop a function named weather_observation_to_dict that takes a tuple representing a day's weather observation and returns a dictionary with keys for "temperature", "humidity", and "wind_speed".
